# Simply News App

## Installation

1. Clone Repo
```
git clone https://github.com/LukaszKwiatkowski94/Simply-news-app.git
```

In the main directory

2. Install composer
```
composer install
```

3. Update the autoloader
```
composer dump-autoload
```

4. Run server
```
composer run-server
```

5. Go to page
```
http://localhost:8800
```
